# Password Generator 🔐

A Python-based Password Generator built as part of the Oasis Infobyte Python Development Internship (AICTE OIB-SIP).

## 📌 Project Description
This project generates a strong random password based on user preferences. Users can customize the length and character types included in the password. It includes both a CLI and GUI version.

## 🚀 Features
- Customizable password length
- Choose to include letters, numbers and symbols
- Generates a strong random password instantly
- Handles invalid inputs and negative values gracefully
- Color coded result display (GUI version)

## 🛠️ Tech Stack
- Python 3
- Tkinter (GUI)

## ▶️ How to Run

### CLI Version
1. Clone this repository
   git clone https://github.com/amanverma123-prog/OIBSIP.git
2. Navigate to the project folder
   cd OIBSIP/Password_Generator
3. Run the script
   python password_generator.py

### GUI Version
1. Navigate to the project folder
   cd OIBSIP/Password_Generator
2. Run the GUI script
   python password_generator_gui.py

## 📁 Project Structure
OIBSIP/
└── Password_Generator/
    ├── password_generator.py      ← CLI version
    └── password_generator_gui.py  ← GUI version (Tkinter)

## 🖥️ GUI Version
Built using Tkinter with:
- Input field for password length
- Checkboxes for character type selection
- Color coded result display
  - 🟢 Green → Password generated successfully
  - 🔴 Red → Error or invalid input
