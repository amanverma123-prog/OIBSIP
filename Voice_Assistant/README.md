# Voice Assistant 🎤

A Python-based Voice Assistant built as part of the Oasis Infobyte Python Development Internship (AICTE OIB-SIP).

## 📌 Project Description
This project is a voice-controlled assistant that listens to your commands and responds accordingly. It can open websites, tell the time and date, and search Wikipedia.

## 🚀 Features
- 🎤 Voice recognition using SpeechRecognition
- 🔊 Text to speech using pyttsx3
- 🌐 Open YouTube and Google
- 📖 Wikipedia search
- 🕐 Tell current time and date
- 👋 Greets user based on time of day

## 🛠️ Tech Stack
- Python 3
- SpeechRecognition
- pyttsx3
- Wikipedia
- webbrowser
- datetime

## ▶️ How to Run
1. Clone this repository
   git clone https://github.com/amanverma123-prog/OIBSIP.git
2. Install required libraries
   pip install speechrecognition pyttsx3 wikipedia pyaudio
3. Navigate to the project folder
   cd OIBSIP/Voice_Assistant
4. Run the script
   python voice_assistant.py

## 🗣️ Supported Commands
| Command | Action |
|---|---|
| "open youtube" | Opens YouTube |
| "open google" | Opens Google |
| "wikipedia [topic]" | Searches Wikipedia |
| "what is the time" | Tells current time |
| "what is the date" | Tells current date |
